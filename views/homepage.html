OctoGuard

Defend your GitHub repos from spam, automatically.

Areas protected

  • Commit comments
  • Issues and Pull Requests
  • Isolate watchers, stargazers and forks beforehand

Actions

  • Lock conversations
  • Delete/edit comments
  • Leave predefined replies
  • Apply labels
  • Block users (need additional OAuth integration)
